# 702 Project Public
THE 702 PROJECT

Project page for the 702 Project. 

This is a personal project about a 11 inch 2-in-1 Windows/iPad OS laptop. 
The project started in 2022 with the hope of creating a handy small laptop, which slowly turned into a project making the ideal laptop in my mind. To be the ideal, it needs to have the best hardware and software the current world could offer, which led to the idea of combining an iPad, which houses the best monitor, webcam, speaker, microphone, and battery efficiency on consumer electrics, and an x86 motherboard offering versatility and capabilities. (This is definitely not because I couldn't wire an 11 inch IPS display onto the Framework mainboard) The Framework 13 mainboard will be used as motherboard because it was the only open source motherboard available at the time with the best performance figure and upgradability. The butterfly keyboard was used because first, I love ThinkPads and the one on 701C was too classic to not replicate; and second, there really aren't much space for a decent sized keyboard. A plam rest was later added after the first keyboard prototype was made in 2024 for better ergonomics. The palm rest is also in sync with the motion of the keyboard. 

The project is currently under construction and in the early stage of concept proving and prototyping. Everything is subject to change so I only open sourced the files I am sure they worked. Any critics are welcomed!

Configuration:
A side: iPad (monitor, webcam, speaker, microphone) [ARM side]
B side: Framework 13 mainboard + battery (keyboard, mouse, motherboard, battery) [x86 side]

Usage:
When using the laptop you could choose between the 2 devices/systems. For note taking, sketching, reading, or anything more handy on the iPad, you could take the lid off and use it as an iPad. If tasks require a Windows environment, plug the iPad back on to the base and proceed to use as a normal laptop. 


Currently the laptop is still under construction. Progress will be posted below.
